## Daily_Study

### HTML 문서의 기본 구조
 &lt;html&gt;는 웹 문서의 시작을 알리는 html 태그\
 &lt;head&gt;는 웹 브라우저에 문서 정보를 알려주는 태그\
 &lt;title&gt;는 문서 제목을 나타내는 태그\
 
### HTML 첫 문서 연습하기
<!DOCTYPE html>
<html lang="ko"> 
<head>
  <meta charset="UTF-8">
  <title>HTML 첫 번째 문서 연습</title>
</head>
<body>
  <h1>웹 문서 만들기</h1>
</body>
</html>
